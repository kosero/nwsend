# nwsend [![Licence](https://img.shields.io/badge/License-BSD-red.svg)](LICENSE)

# install
```bash
git clone https://github.com/kosero/nwsend.git
cd nwsend && go build .
mkdir $HOME/.local/bin
mv ./nwsend .local/bin/
```

# config
example config
```json
{
    "name": "kosero",

}
```
